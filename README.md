## TripleD - A Webpage for our company focused on 3D printing solutions.

This is a NextJS project.

# Getting started
- Install depedencies with yarn

- Run a local dev server with
```
yarn dev
```

- Preview a production server
```
yarn start
```

- Build the project
```
yarn build
```

# Hosting
